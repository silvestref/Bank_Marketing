# Bank_Marketing
Implementación de la metodología de ciencia de datos para el análisis de datos y la construcción de un modelo de machine learning basado en CatBoost para la identificación y el entendimiento de patrones que ayuden a predecir que clientes solicitarán un depósito a plazo fijo en la próxima campaña anual de un banco portugués, asi como el comportamiento que
presentan aquellas personas que solicitaron o no dicho servicio.
